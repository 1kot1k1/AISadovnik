# 🌿 Plant Disease Detection System

**Python • TensorFlow • Flask • ResNet50 • MobileNetV2 • Computer Vision**

Plant Disease Detection System — интеллектуальная система компьютерного зрения для автоматического распознавания заболеваний растений по изображениям листьев. Проект объединяет технологии глубокого обучения и веб-разработки, позволяя пользователю выполнять диагностику растений через удобный браузерный интерфейс.

---

## 🚀 Ключевые возможности

### 🤖 Двойная система распознавания

Приложение одновременно использует две современные архитектуры нейронных сетей:

* **ResNet50**
* **MobileNetV2**

Это позволяет сравнивать результаты различных моделей на одном изображении и анализировать их точность.

### 🔍 Автоматическая диагностика заболеваний

После загрузки фотографии система:

* выполняет предобработку изображения;
* запускает анализ через обученные нейронные сети;
* определяет наиболее вероятное заболевание;
* отображает уровень уверенности модели;
* формирует Top-5 наиболее вероятных диагнозов.

### 🌱 Поддержка нескольких культур

Модель обучена на датасете **PlantVillage** и поддерживает диагностику:

* Болезней томатов;
* Болезней картофеля;
* Болезней сладкого перца;
* Здоровых растений.

Всего реализовано **15 классов заболеваний и состояний растений**.

### 🌐 Веб-интерфейс

Пользователь может:

* загружать изображения через браузер;
* получать результаты анализа в режиме реального времени;
* сравнивать результаты нескольких моделей;
* использовать систему без установки дополнительного программного обеспечения.

## 🧠 Архитектура системы

### Модели машинного обучения

Каждая модель использует подход **Transfer Learning** на основе предварительно обученных весов ImageNet.

#### ResNet50

Особенности:

* Residual Connections;
* высокая устойчивость к переобучению;
* высокая точность классификации;
* эффективная работа с глубокими сетями.

#### MobileNetV2

Особенности:

* облегчённая архитектура;
* высокая скорость инференса;
* низкие требования к памяти;
* оптимизация для веб- и мобильных решений.

## 💻 Запуск проекта

### Установка зависимостей

```bash
pip install tensorflow flask pillow numpy
```

### Подготовка моделей

Поместите в папку `models` следующие файлы:

* `resnet_weights.weights.h5`
* `mobilenet_weights.weights.h5`
* `class_names.npy`

### Запуск приложения

```bash
python app.py
```

После запуска откройте браузер:

```text
http://127.0.0.1:5000
```
